# Инструкция по настройке шаблона

## Шаг 1: Подготовка изображения-шаблона

### Если у вас есть PDF-файл:

1. Конвертируйте PDF в PNG с нужным размером
   - Размер должен быть: **797 × 1132 пикселей** (вертикальная ориентация)
   - Используйте: Adobe Acrobat, Inkscape, или онлайн-конвертер
   - Или через командную строку: `convert -resize 797x1132! input.pdf assets/template.png`

2. Сохраните результат как `assets/template.png`

### Если у вас уже есть PNG:

1. Убедитесь, что размер изображения: **797 × 1132 пикселей** (или пропорционально, но тогда нужно пересчитать координаты)
2. Поместите файл в `assets/template.png`

### Текущая разметка шаблона:

```
┌─────────┬─────────┬─────────┐
│   ПН    │   ВТ    │   СР    │ (верхняя часть, y: 140)
├─────────┴─────────┬─────────┤
│        ЧТ         │   ПТ    │ (нижняя часть, y: 360)
└───────────────────┴─────────┘
```

**Важно:** Блоки ГАРНИРЫ/НАПИТКИ (справа снизу) не используются для меню дней

## Шаг 2: Определение координат зон

Для каждого дня недели необходимо определить зону размещения текста:

- **x, y** - координаты левого верхнего угла зоны
- **width** - ширина зоны в пикселях
- **max_height** - максимальная высота, которую может занимать текст

### Как найти координаты:

1. Откройте изображение в графическом редакторе (GIMP, Photoshop, Figma)
2. Используйте инструмент "Линейка" или "Информация о пикселях"
3. Определите координаты для каждого дня

### Текущие координаты (для шаблона 797×1132 px):

```
ПН: x=55,   y=148, width=330, max_height=360  (левая колонка, верх)
ВТ: x=55,   y=546, width=330, max_height=297  (левая колонка, середина)
СР: x=55,   y=857, width=330, max_height=228  (левая колонка, низ)
ЧТ: x=410,  y=218, width=300, max_height=332  (правая колонка, верх)
ПТ: x=410,  y=601, width=300, max_height=207  (правая колонка, низ)
```

**Примечание:** Координаты указаны в пикселях от верхнего левого угла

## Шаг 3: Обновление zones.yaml

Координаты для шаблона A4 уже настроены в `config/zones.yaml`.

Если вы используете другой шаблон, отредактируйте файл:

```yaml
ПН:
  x: 30
  y: 140
  width: 110
  max_height: 190
```

**Важно:** При изменении размера шаблона или DPI конвертации пересчитайте координаты пропорционально!

## Шаг 4: Настройка шрифтов

1. Поместите файл шрифта (TTF или OTF) в `assets/fonts/`
2. Обновите пути в `config/settings.yaml`:

```yaml
fonts:
  title:
    file: assets/fonts/Inter.ttf
    size: 10.05
```

## Шаг 5: Тестирование

1. Запустите бота
2. Отправьте тестовое меню
3. Проверьте результат
4. При необходимости скорректируйте координаты или размеры шрифтов

## Советы

- Оставляйте небольшой запас по высоте (не заполняйте `max_height` на 100%)
- Если текст не помещается, уменьшите размер шрифта или `line_spacing`
- Используйте контрастные цвета текста относительно фона шаблона
- Тестируйте с разным количеством блюд в день
